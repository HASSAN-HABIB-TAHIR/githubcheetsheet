```bash Clone a repository: git clone [repository URL]

Create a new branch: git branch [branch name]

Switch to a branch: git checkout [branch name]

Add changes: git add .

Commit changes: git commit -m "[commit message]"

Push changes: git push [remote] [branch name]

Pull changes: git pull [remote] [branch name]

Merge changes: git merge [branch name]

View status: git status

View commit history: git log

Create a tag: git tag [tag name]

Push a tag: git push [remote] [tag name]

Delete a tag: git tag -d [tag name]

View all tags: git tag

Delete a branch: git branch -d [branch name]

View all branches: git branch ```
